# TaskFlow API

A RESTful Backend API built using Node.js and Express.js.

## Features

- Create Task
- Get All Tasks
- Get Single Task
- Update Task
- Delete Task
- Error Handling
- Middleware
- Security Headers

## Technologies Used

- Node.js
- Express.js
- CORS
- Morgan
- Helmet

## API Endpoints

### GET All Tasks

GET /api/tasks

### GET Single Task

GET /api/tasks/:id

### POST Create Task

POST /api/tasks

### PUT Update Task

PUT /api/tasks/:id

### DELETE Task

DELETE /api/tasks/:id

## Run Project

npm install

npm run dev
